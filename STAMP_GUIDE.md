## 追加手順

1. `stamps` フォルダを開く
2. 画像を `10.png`, `20.png`, `30.png` ... `400.png` の名前で保存する
3. GitHub Pagesへアップロードする
4. iPad側でアプリ内の「最新版に更新」を押す

`stamps/stamps.js` の条件は上記の40個に合わせてあります。画像名を変える場合は、`stamps/stamps.js` の `src` も同じ名前に変更してください。
